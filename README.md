# prismagram

Instagram clone with Express + Prisma + React and React Native

## User Stories

- [ ] Create account
- [ ] Log in
- [ ] See the feed
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search by location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
